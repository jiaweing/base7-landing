### Local Development

1. Clone the repository

   ```bash
   git clone https://github.com/jiaweing/base7-landing.git
   cd base7-landing
   ```

2. Install dependencies

   ```bash
   bun install
   ```

3. Start the development server

   ```bash
   bun dev
   ```
